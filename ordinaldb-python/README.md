# OrdinalDB Python

Python bindings and optional framework adapters for OrdinalDB.

The base package exposes local OrdVec-backed vector indexes without importing
framework integrations. Optional extras provide LangChain, LlamaIndex,
Haystack, and Agno adapters.

See the repository README for full project documentation:
https://github.com/Project-Navi/ordinaldb

For edge/local adapter deployment, including offline wheelhouse setup, valid
embedding dimensions, explicit save boundaries, and telemetry controls, see:
https://github.com/Project-Navi/ordinaldb/blob/main/docs/edge-deployment.md
